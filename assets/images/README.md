# 📁 Panduan Folder Gambar

Ganti file placeholder di setiap folder dengan foto asli kamu.
**Format yang disarankan:** JPG atau WebP
**Ukuran maksimal:** di bawah 500KB per gambar agar website cepat

---

## 📂 hero/
**Nama file:** `hero-main.jpg`
**Ukuran ideal:** 1600 × 900 px (landscape)
**Isi:** Foto teknisi sedang bekerja / spraying, background gelap

---

## 📂 about/
**Nama file:** `team-about.jpg`
**Ukuran ideal:** 800 × 600 px
**Isi:** Foto tim GreenShield berpose profesional / di depan kendaraan

---

## 📂 services/
**Nama file (urutan sesuai tampilan website):**
| File | Layanan |
|------|---------|
| `service-rayap.jpg` | Pengendalian Rayap |
| `service-kecoa.jpg` | Pengendalian Kecoa |
| `service-tikus.jpg` | Pengendalian Tikus |
| `service-nyamuk.jpg` | Pengendalian Nyamuk |
| `service-lainnya.jpg` | Layanan Lainnya |

**Ukuran ideal:** 400 × 300 px (landscape)

---

## 📂 gallery/
**Nama file:** `gallery-1.jpg`, `gallery-2.jpg`, ... dst
**Ukuran ideal:** 600 × 450 px
**Tips:** Foto bisa ditambah dengan menambahkan item baru di `index.html`
pada section galeri dengan tag `<div class="gallery-item">` baru.

**Kategori untuk filter:**
- `data-category="rumah"` → Perumahan
- `data-category="kantor"` → Perkantoran
- `data-category="restoran"` → Restoran
- `data-category="hotel"` → Hotel
- `data-category="gudang"` → Gudang
- `data-category="pabrik"` → Pabrik

---

## 📂 testimonials/
**Nama file:** `avatar-1.jpg`, `avatar-2.jpg`, `avatar-3.jpg`
**Ukuran ideal:** 200 × 200 px (persegi / square)
**Isi:** Foto profil klien
